# VideoReapter
其于electron的全平台音视频复读机，运行于windows, mac, linux平台。注意：media目录中只保存了字幕文件，并无音视频文件。

## 功能
- 设定复读次数
- 中英文切换
- 自动保存学习进度（数据保存在本地的media.json中）

## 安装
```
git clone https://github.com/dayudodo/VideoRepeater.git
cd VideoRepeater
npm install
```

## 编译
```
npm run build
```

## 运行
```
npm run start
```
